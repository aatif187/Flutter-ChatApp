# Chat_App

A Real-Time Chatting Mobile Application. Easily search any user and chat. SignUp amd SignIn smoothly.

## Features-
- Real-Time Chatting
- Searching any User
- User Authentication (sign Up, Sign In)

## Techs/Frameworks used-
- Flutter
- FireBase
- Dart (language)
